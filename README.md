# Beat Runner Class Diagrams

Class Diagram 均由沈威宇使用 clang-uml 自原始 C++ 程式碼中自動萃取類別結構資訊，再依實際設計意圖進行人工補註與整理，最後透過 PlantUML 生成。

## 圖例

### class 成員符號

- 實心方形：private method
- 空心方形：private field
- 實心菱形：protected method
- 空心菱形：protected field
- 實心圓形：public method
- 空心圓形：public field

### class 連結箭頭

- 實線空心三角箭頭：derive from（inheritance）
- 虛線空心三角箭頭：be realized with（interface realization）
- 實線開口箭頭：composite（composition）
- 虛線開口箭頭：depend on（dependency）
- 實線空心菱形箭頭：aggregate in（aggregation）
- 實線 ⊗ 形箭頭：be aliased by（type alias via `using`）
